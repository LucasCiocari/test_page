Izuku Midoriya

Apesar de ter nascido sem nenhum dom, sempre quis ser um herói como o All Might. Para isso, ele estudou os poderes dos heróis profissionais assim como os vilões. Depois de ser salvo por All Might, Midoriya pergunta para ele se pode se tornar um herói mesmo que não tenha nenhum dom. Comovido, All Might diz que vai passar para ele o poder do One For All, que é o que lhe garante a super força.
